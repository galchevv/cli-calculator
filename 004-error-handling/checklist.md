## Самопроверка: финальный прогон

Открой терминал. Без подсказок прогони эти команды и убедись, что результат правильный:

```bash
go run main.go               # подсказка по использованию, exit 1
go run main.go 1 + 1         # Результат: 2, exit 0
go run main.go foo + 1       # ошибка про "foo", exit 1
go run main.go 1 + bar       # ошибка про "bar", exit 1
go run main.go 1 / 0         # ошибка про деление на ноль, exit 1
go run main.go 1 ? 1         # ошибка про неизвестную операцию, exit 1
go run main.go 100 - 50      # Результат: 50, exit 0
```

Если все 7 случаев работают как ожидается — поздравляю, ты прошёл самый важный концептуальный модуль этого проекта. **Идиома `if err != nil` теперь твоя на всю жизнь.**
