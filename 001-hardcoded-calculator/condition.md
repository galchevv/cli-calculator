# Модуль 1. Захардкоженный калькулятор — условие

## Задание

Напиши программу `main.go`, которая выполняет четыре арифметические операции с двумя числами, **жёстко прописанными в коде**, и выводит результаты на экран.

## Что должна делать программа

При запуске командой `go run main.go` программа должна вывести:

```
Первое число: 20
Второе число: 4
Сумма: 24
Разность: 16
Произведение: 80
Частное: 5
```

## Требования

1. Числа `20` и `4` должны быть **сохранены в переменных** (не подставлены прямо в `fmt.Println`).
2. Результаты операций тоже должны быть **в отдельных переменных** перед выводом.
3. Используй короткое объявление `:=`.
4. Программа должна корректно компилироваться и запускаться без ошибок.

## Шаги (если совсем не знаешь, с чего начать)

1. Создай папку для проекта и зайди в неё в терминале
2. Выполни `go mod init calculator` — это создаст файл `go.mod`
3. Создай файл `main.go`
4. Напиши скелет: `package main`, `import "fmt"`, `func main() { }`
5. Внутри `main` объяви переменные с числами
6. Посчитай результаты в новых переменных
7. Выведи всё через `fmt.Println`
8. Запусти `go run main.go` и проверь

## Эксперименты после того, как всё работает

Когда основная программа заработает, попробуй:

1. **Поменяй числа** на `20` и `3`. Что выведет частное? Почему?
2. **Поменяй тип** переменных на `float64` (числа должны быть `20.0` и `3.0`). Что теперь выведет частное?
3. **Добавь пятую операцию** — остаток от деления (оператор `%`). Только для целых чисел.

Эти эксперименты — самая ценная часть модуля. Они дадут тебе интуицию, которую невозможно получить просто чтением.
